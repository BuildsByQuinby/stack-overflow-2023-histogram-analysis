# Stack Overflow 2023 Histogram Analysis

This project analyzes the 2023 Stack Overflow Developer Survey using histograms to explore salary distribution, experience levels, job satisfaction, technology preferences, and remote work trends.

The project demonstrates data extraction from a SQL database, data cleaning, and visualization using Python.

---

## Dataset

Source: Stack Overflow Developer Survey 2023  
https://www.kaggle.com/datasets/stackoverflow/stack-overflow-2023-developers-survey

Due to file size limits, the raw dataset is not included in this repository.

---

## Objectives

- Visualize salary distributions
- Compare compensation by age group
- Analyze job satisfaction by experience
- Explore remote work preferences
- Identify most desired databases

---

## Tech Stack

- Python
- Pandas
- SQLite
- Matplotlib
- Seaborn
